# 3D model for “Explore the tracker” section

The viewer loads **`exploding-view.glb`** from this folder.

## Export from your Blender file

1. Open **ExplodingView3_10.blend** in Blender.
2. **File → Export → glTF 2.0 (.glb/.gltf)**.
3. Save as **`exploding-view.glb`** in this folder (`public/models/`).

So the file path in the project should be: **`public/models/exploding-view.glb`**.

The homepage section will then show your exploding-view model; users can drag to rotate and scroll to zoom.
